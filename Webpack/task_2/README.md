# Holberton Webpack Task 2

This project demonstrates advanced Webpack configuration for handling CSS and image assets, including optimization, for Holberton School.

## Features
- CSS and image support in Webpack
- Image optimization
- All assets bundled in `public/`
- Production mode for optimal performance

## Usage
1. Install dependencies:
   ```sh
   npm install
   ```
2. Build the project:
   ```sh
   npm run build
   ```
3. Open `public/index.html` in your browser.

## Requirements
- Node 20.x.x
- npm 9.x.x
